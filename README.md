# Ablauf

Durable workflow engine for Cloudflare Workers, powered by Durable Objects.

## Install

```bash
bun install
```

## Run

Run the full monorepo in dev mode:

```bash
bun run dev
```

Run only the demo Worker:

```bash
bun --cwd apps/worker run dev
```
